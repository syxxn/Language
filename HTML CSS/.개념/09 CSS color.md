# CSS color

CSS에서 색을 표현하는 방식이 세가지가 있다.



### 1. 색이름

CSS에서 정해준 색 이름 중 하나를 쓰는 방법이다. 모든 색이 있지는 않지만 140개의 색이 대부분 브라우저에서 지원된다.

```css
h1 {
    color: blue;
}
```



### 2. RGB 값

모든 색은 빨강, 초록, 파랑의 조화로 표현할 수 있다. 

```css
h1 {
    color: rgb(12,10,60);
}
```



### 3. HEX 값 (16진법)

HEX는 단순히 RGB 값을 16진법으로 표현한 방식이다.

```css
h1 {
    color: #2F391A
}
```
